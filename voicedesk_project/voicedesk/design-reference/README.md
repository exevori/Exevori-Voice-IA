1. **Ouvre `dashboard-reference.png`** — niveau de qualité attendu.
2. **Ouvre `branding/exevori-voice-ia-target.png`** — branding officiel à reproduire.
3. **Lis `docs/EMERGENT-DESIGN.md`** — breakdown complet + briques GitHub.
4. **Lis `docs/EMERGENT-WORKSHEET.md`** — feuille de travail fichier-par-fichier.
